Css Combinators

  1. Decendant (space)
    Selects all d elements inside a parent
  2. Child (>)
    Selects direct child only inside a parent
  3. Next Sibling (+)
    selects first element comes after the mentioned element
  4. Subsequent-Sibling (~)
    selects all the element after the first child

Pseudo Class

Used to define a special state of an element

        selector:pseudo-class {
            property: value;
          }
          Example

          a:link{
            color:red;
          }
          a:hover{} a:active a:visited a:hover

div p:first-child matches first element of every div

Pseudo element

Is used to style specific parts of the element

            selector::pseudo-element {
                property: value;
              }


              Example
              p::first-line {
                color: #ff0000;
                font-variant: small-caps;
              }
              p::first-letter {
                color: #ff0000;
                font-size: xx-large;
              }
              h1::before {
                content: url(smiley.gif);
              }
              h1::after {
                content: url(smiley.gif);
              }

              ::selection {
                color: red;
                background: yellow;
              }
